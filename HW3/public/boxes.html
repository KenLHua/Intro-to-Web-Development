Floating

This is some text that will float because of the image. We can display content that is offset due to other elements.

If we don't want this effect, we can use the attribute clear.

If we want to offset an element from its static positioning, we can use relative positioning.

We can also have fixed elements that stay on view, we can use fixed positioning.

Using z-index can allow us to describe which content should be overlapped, and which should do the overlapping.
